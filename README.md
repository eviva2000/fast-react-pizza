# 🍕 Fast React Pizza Co.

A modern, fast, and responsive pizza ordering application built with React. This project demonstrates advanced React patterns, state management with Redux Toolkit, and sophisticated routing with React Router.

## 🚀 Features

- **Menu Browsing**: Explore a wide variety of pizzas with real-time availability.
- **Cart Management**: Seamlessly add, remove, and update pizza quantities in your cart.
- **Ordering System**: Secure checkout process with order tracking.
- **User Profiles**: Personalized experience with saved user names.
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices using Tailwind CSS.

## 🛠️ Technologies Used

- **React 18**: Core UI library.
- **Vite**: Ultra-fast frontend build tool.
- **React Router 6**: Declarative routing for single-page applications.
- **Redux Toolkit**: Efficient and predictable state management.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **ESLint & Prettier**: For code quality and consistent formatting.

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
